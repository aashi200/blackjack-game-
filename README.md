# Blackjack Game

A browser-based implementation of the classic Blackjack card game built using JavaScript.

## Features

* Random card generation
* Game logic for Blackjack rules
* Player vs dealer gameplay
* Dynamic score updates

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository
2. Open `index.html` in a browser
3. Start playing Blackjack
